En Whatsapp y Telegram, cuando pegas un bloque de código para hacer una consulta resulta muy poco legible. En Discord, solo hay que abrir el bloque de código con tres acentos invertidos al inicio, el nombre del lenguaje, luego el bloque de código y finalmente cerrar con otros tres acentos invertidos: Ejemplo con HTML:
<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Page Title</title> </head> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> <p>Que grande <a href="http://www.discord.com/">Discord</a>!</p> </body> </html>

# función que indica si una cadena es igual a su inverso def esPalindromo(id:int)->bool: s = str(id) print(s,s[::-1]) return s == s[::-1] # Ingreso de datos id = input("Ingrese un id (100 a 999): ") respuesta = esPalindromo(id) if respuesta: print("Sí") else: print("No")
